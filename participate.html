Participate

Follow the steps below to participate in the MaxCLIP competition:

  1. Clone and Modify the FastCLIP Repository

    Start by cloning the FastCLIP base repository to make your changes. This repository provides the starting point for your models:

    FastCLIP GitHub Repository

  2. Clone the DataComp Repository for Evaluation

    For evaluating your model, clone the DataComp repository which includes the necessary evaluation scripts:

    DataComp GitHub Repository

  3. Submit Your Model for Evaluation

    Once you have made your changes and evaluated your model, package your submission and send it to us for evaluation:

    Email your submission to: submit@maxclip.net
